## 03 — ExamPilot
**Problem:** Engineering entrance exam students fail not from lack of effort, but from studying the wrong things in the wrong order — with no feedback loop.
**Proposed Solution:** An AI-powered performance optimisation system built on a Student Digital Twin that eliminates wasted study time and makes score outcomes predictable.
**Includes:** 4-layer system architecture, full feature specification, UX philosophy (3-tap rule), MVP-to-full-product roadmap, competitive analysis, tech stack, failure warnings.
